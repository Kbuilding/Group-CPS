# Pseudocode for Cybernetics + Public Art CPS  

Pseudocode is used to communicate an algorithm to anyone with basic programming knowledge.

Here is the pseudocode for our CPS:

    IMPORT Dependencies/ Libraries 
    IMPORT CSV File
